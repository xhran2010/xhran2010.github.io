旋转图像

给定一个 n × n 的二维矩阵表示一个图像。

将图像顺时针旋转 90 度。

说明：

你必须在原地旋转图像，这意味着你需要直接修改输入的二维矩阵。请不要使用另一个矩阵来旋转图像。

示例 1:

给定 matrix =
[
[1,2,3],
[4,5,6],
[7,8,9]
],

原地旋转输入矩阵，使其变为:
[
[7,4,1],
[8,5,2],
[9,6,3]
]

按层旋转

这也是一开始我想到的方法，整张图片的旋转可以看做是每一层旋转90度；每一层的旋转又可以看做是每一个边旋转90度；每一个边旋转又可以看做是边上的每个点旋转90度。只是我一开始想的是把每个边从到到尾处理，但这样图像的顶角就会出现重复，而官方题解中给了如下的图：

进一步像这样优化之后，就可以设置两层循环，时间复杂度为$O(n^2)$。第一层循环为遍历层数，第二层循环为遍历每一层边上的每个点。代码量很小，主要就是分析清楚边上的每个点怎么用循环变量来表示。

参照以下代码，第一层循环变量l为层数，起始为第0层，截止为matrix.size() / 2。第二层循环变量c为边上的每个点，遍历的长度即为边的长度。而经过上图的划分后，每个边的长度为matrix.size() - 1 - l * 2（因为每向里面进一层，左右边缘就要分别少一格，总的就会少两格）
